---

## ■ 소개

사용자의 위치 기반으로 다양한 여행가이드를 제공합니다. Kakao 또는 Google 소셜 로그인을 통하여 게시물의 좋아요 및 댓글을 작성 할 수 있으며, 실시간 채팅 기능을 지원합니다.
